# Low‑fi WIREFRAME — «Техника событий»

Это кликабельный web‑прототип (low‑fi wireframe) со структурой и переходами по роутам из задания.

## Как открыть локально

Рекомендуется запускать через локальный сервер (так корректно работают ссылки вида `/uslugi/...`).

```bash
cd "/Users/annaslnv/Documents/курсор/tekhnika-event-wireframe"
python3 -m http.server 8000
```

Открыть в браузере: `http://localhost:8000/`
